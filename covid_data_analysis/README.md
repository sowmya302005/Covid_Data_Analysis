# COVID-19 Data Analysis Project

Analyzed COVID-19 data to track trends in total cases, daily new cases, and deaths using Python.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Identified peak periods of COVID-19 cases
- Tracked daily case trends
- Visualized long-term impact through line charts

## Dataset
- Kaggle / Our World in Data
